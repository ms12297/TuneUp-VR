Tune Up

Tune up is a single player 3-D browser based game that challenges the reaction time of the player using engaging sound and visuals. The goal of the game is to get up to 20 points to win. Player gets points by standing on the right tile and clicking the right instrument that's displayed in front of them. The player loses if they lose 3 consecutive points in a row.

Instructions

  • Using the IJKL keys, the user is able to teleport to the middle of each colored tile.
  • The user needs to stand on the tile corresponding to the color of the moving tile.
  • The railings change colors; green for a good match, red for a false match.
  • The user is prompted to click on the corresponding displayed object. A correct click results in a beat, a wrong click will ruin the melody.
  • The game ends when three consecutive points are lost (loss) or if the score is greater than 20 (win).
